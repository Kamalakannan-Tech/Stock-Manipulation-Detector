# API Documentation

The Stock Manipulation Detection System exposes a RESTful API powered by Flask. The API provides access to stock metadata, model predictions, real-time alerts, and system health status.

## Base URL
\`http://localhost:5000\`

---

## Endpoints

### 1. System Health
Check if the API and underlying services are running.

- **URL:** \`/health\`
- **Method:** \`GET\`
- **Success Response:**
  - **Code:** 200 OK
  - **Content:**
    ```json
    {
      "status": "healthy",
      "timestamp": "2026-05-12T10:00:00"
    }
    ```

---

### 2. Get Monitored Stocks
Retrieve a list of all stock tickers currently being monitored by the system.

- **URL:** \`/api/stocks\`
- **Method:** \`GET\`
- **Success Response:**
  - **Code:** 200 OK
  - **Content:**
    ```json
    [
      "GME",
      "AMC",
      "TSLA",
      "AAPL",
      "MSFT",
      "NVDA"
    ]
    ```

---

### 3. Get Stock Details
Retrieve detailed information and the latest metrics for a specific stock.

- **URL:** \`/api/stocks/<ticker>\`
- **Method:** \`GET\`
- **URL Parameters:**
  - \`ticker\` (string): The stock symbol (e.g., \`GME\`).
- **Success Response:**
  - **Code:** 200 OK
  - **Content:**
    ```json
    {
      "ticker": "GME",
      "company_name": "GameStop Corp.",
      "current_price": 24.50,
      "volume": 15400000
    }
    ```

---

### 4. Get Stock Prediction
Run the inference engine or retrieve the latest cached manipulation prediction for a specific stock.

- **URL:** \`/api/stocks/<ticker>/prediction\`
- **Method:** \`GET\`
- **URL Parameters:**
  - \`ticker\` (string): The stock symbol.
- **Success Response:**
  - **Code:** 200 OK
  - **Content:**
    ```json
    {
      "ticker": "GME",
      "risk_level": "medium",
      "risk_score": 45.2,
      "manipulation_probability": 0.452,
      "model_version": "v1.0",
      "timestamp": "2026-05-12T10:05:00"
    }
    ```

---

### 5. Get Alerts
Retrieve a list of recent manipulation alerts generated by the system.

- **URL:** \`/api/alerts\`
- **Method:** \`GET\`
- **Query Parameters (Optional):**
  - \`limit\` (integer): Number of alerts to return (default: 50).
- **Success Response:**
  - **Code:** 200 OK
  - **Content:**
    ```json
    [
      {
        "alert_id": "5f8d0a9b",
        "ticker": "AMC",
        "risk_level": "high",
        "message": "Unusual social volume detected alongside price spike.",
        "timestamp": "2026-05-12T09:45:00"
      }
    ]
    ```

---

### 6. Get Analysis Overview
Retrieve a comprehensive analysis report for a stock, including technical indicators and social sentiment summary.

- **URL:** \`/api/analysis/<ticker>/overview\`
- **Method:** \`GET\`
- **URL Parameters:**
  - \`ticker\` (string): The stock symbol.
- **Success Response:**
  - **Code:** 200 OK
  - **Content:**
    ```json
    {
      "ticker": "GME",
      "technical_analysis": {
        "rsi_14": 72.5,
        "macd": 1.2
      },
      "social_sentiment": {
        "score": 0.85,
        "mention_volume": 15000
      }
    }
    ```

---

## WebSockets
The system also supports WebSocket connections for pushing real-time alerts and stock updates to the dashboard. Connect to the WebSocket server (typically running on the same port or via a dedicated Socket.io namespace) to listen for \`alert\` and \`price_update\` events.
